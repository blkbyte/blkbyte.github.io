# 🌐 blkbyte Portfolio (В стиле bios)

Сайт-портфолио, выполненный в стиле **BIOS**, написан на **HTML**, **CSS** и немного **JavaScript**.

В будущем проект будет полностью переписан на React для более динамичного интерфейса.

![main](/assets/website.png) 

## 💡 Главная идея

Главная идея сайта — нижний терминал, который имитирует консоль. Пока он не идеален, но это основная концепция.

## 🔢 Счётчик посещений
Счётчик реализован с помощью [count.getloli.com](https://count.getloli.com)

## ⚡ Как запустить проект

```bash
# Клонируем репозиторий
git clone https://github.com/blkbyte/blkbyte.github.io
cd blkbyte.github.io

# Откройте index.html в браузере
```
## 🤝 Можно ли использовать код?

Да, вы можете использовать этот проект, но пожалуйста, оставьте упоминание автора: blkbyte.
